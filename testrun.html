  • git add .

    adds all the changes in current directory to repository

  • git commit -m 'someString'

    commits changes to repository with the message of 'someString'

  • git push origin master

    pushes repository changes up to whatever location is set to origin, in specific the master branch

  • git merge branchName

    Merges changes from branchName to current working branch, may cause merge conflicts

  • git pull origin master

    pulls changes from origin's master branch and merges them automatically to working branch
